## 测试说明

event_collector 通过 Boost.Asio 异步 TCP 接收事件，经 BoundedQueue 中转后由 4 个 Worker 线程处理。
当前测试模式：Collector + Processor（未启用 Kafka/ClickHouse/Redis，仅入队 + 解析 + 校验路径）。
